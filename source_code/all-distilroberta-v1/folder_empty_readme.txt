This folder contains the embedding model "all-distilroberta-v1" used for RAG.
This model is used in the RAG mechanism with ECLASS library
However, RAG is not used in this uploaded code due to ECLASS coypright.
So, this folder is emptied for the uploaded code, because the model size is too large.
The model can be downloaded from the Huggingface model hub: https://huggingface.co/sentence-transformers/all-distilroberta-v1
